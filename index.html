Snippet Viewer

A vanilla web component for displaying syntax-highlighted code snippets from a JSON file.

View Demo

Installation

Add the script to your page:

<script src="https://your-site.netlify.app/snippet-viewer.js"></script>

Usage

Create a snippets.json file:

{
  "my-function@example.ts": "function hello() {\n  console.log('Hello!');\n}"
}

Use the component:

<snippet-viewer
  snippet="my-function@example.ts"
  snippet-host="https://your-site.com/snippets">
</snippet-viewer>

With Provider (Multiple Snippets)

Use a provider to share the host across multiple viewers:

<snippet-provider snippet-host="https://your-site.com/snippets">
  <snippet-viewer snippet="function-a@file.ts"></snippet-viewer>
  <snippet-viewer snippet="function-b@file.ts"></snippet-viewer>
</snippet-provider>

Features

  • Syntax highlighting via Prism.js (loaded automatically)
  • Auto-detects language from file extension
  • Shared cache across all instances
  • Shadow DOM isolation (safe for WordPress)
  • Provider component for multiple snippets
  • Supports TypeScript, JavaScript, Java, Python, and more

Snippet Key Format

Keys use the format: name@filename.ext

  • name - Identifier for the snippet
  • filename.ext - Displayed in header, extension determines syntax highlighting
